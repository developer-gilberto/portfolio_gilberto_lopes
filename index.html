Uma API REST para meu próprio SaaS ShopTurbo. Desenvolvi essa API para fazer a integração e consumo da API oficial da Shopee com fluxo completo de autenticação via OAuth. A API busca os dados da loja Shopee do usuário e retorna para o frontend os dados paginados. Alguns desses dados são: dados da própria loja, dados dos produtos (preço, estoque, sku, etc) e dados de pedidos (produto comprado, quantidade, valor de cada produto, valor total do pedido, etc), entre outros.

ShopTurbo (SaaS) JavaScript Next.js React TailwindCSS

Um sistema SaaS para pessoas que possuem loja na Shopee, com integração com a API oficial da Shopee. Um sistema onde você só precisa informar quanto pagou pelo produto, o imposto e PRONTO! O ShopTurbo possui uma API própria que busca os dados dos produtos da sua loja na Shopee, faz os cálculos necessários e mostra em uma interface intuitiva um relatório completo com os custos (frete, impostos, preço de custo, etc), faturamento e lucro líquido da sua loja Shopee. Chega de vender no escuro e não saber seus custos e lucros reais. Assine o ShopTurbo e tenha controle total da sua loja Shopee!
